## Controls


| Input  | Key     |
| ----   |:-------:|
| Up     | W/UP    |
| Down   | S/DOWN  |
| Left   | A/LEFT  |
| Right  | D/RIGHT |
| Shoot  | SPACE   |

| Input  | Key     |
| ------------------------ |:------:|
| Increase Shooting Delay  | +      |
| Decrease Shooting Delay  | -      |

## Requirements

- Java (1.7)
- Maven (1.3)

## Build Steps

* mvn install
* cd target
* java -jar JavaAsteroidGame-1.0-SNAPSHOT.jar

## Run Steps

* java -jar JavaAsteroidGame-1.0-SNAPSHOT.jar

Have fun!
