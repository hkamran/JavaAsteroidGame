# How to run

mvn install<br />
cd target<br />
java -jar JavaAsteroidGame-1.0-SNAPSHOT.jar<br />

Have fun!

# Controls

UP, DOWN, LEFT, RIGHT, SPACE<br />
alternately..<br />
W, S, A, D, SPACE
